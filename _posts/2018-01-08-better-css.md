### 一些`css`规范

#### 关于`css`属性值简写

1.`margin`、`padding`的简写

> 以下为`margin`的栗子，`padding`相同。

``` css
margin:10px 10px 10px 10px;
margin:10px; /*简写*/

margin:10px 5px 10px 5px;
margin:10px 5px;/*简写*/

margin:10px 5px 15px 5px;
margin:10px 5px 15px;/*简写*/
```
